## JennWithoutAI - My new journey without AI

### Introduction
I'm Jenn and im writing this on April 4th 2026. <br>
I have been a developer for approx 10+ years now and with the big presence of AI, i have noticed my productivity going up, but my mental understanding of code and knowledge going down.<br>

When autocompleting, i noticed myself iterating things over and over again, and not understanding my code.
Thus, I came in a bad spiral and for the past months i had 0 fun, motivation and willpower to code.

It even became so bad that i had gotten in a bit of a burnout over the year.
<br> But the willpower to come back was big, and after watching lots and lots of video's about this happening to other programmers/engineers i decided to go cold turkey on AI.

### Rules i follow for my craft

Previously in the introduction i talked about that i diddnt have willpower to code<br> 
While this is true, my willpower to return to coding grew and grew every day.

So i decided To make rules **to do / not do** certain things while working on projects or in my programming journey :
1. No AI (duh)
2. No AI Line Autocomplete in Tools / Code Editor's
3. Strictly use Documentation 
4. When googling adding -AI to the search (This is awesome btw)
5. Giving myself space to program, so no instant & rapid dopamine **BS** like youtube or tiktok or whatever.
6. Dont beat myself up about my skills, because this is the great Programmer Reset. 
7. Doesnt have to be pretty and perfect the first time.
